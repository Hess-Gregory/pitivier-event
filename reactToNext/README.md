# react-cms
React landing page implement express-cms in other repository.

![Imgur Image](https://imgur.com/LEtD8Yi.png)

## Installation

```sh
git clone https://github.com/muhammadardie/react-cms.git
cd react-cms
npm install
cp .env.example .env
npm start
```
