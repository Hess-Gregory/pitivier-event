# react-cms
React landing page implement express-cms in other repository.

![Imgur Image](https://pitivierevents.hess-gregory.be/pitivier.JPG)

## Installation

```sh
git clone https://github.com/Hess-Gregory/pitivier-event.git
cd react-cms
npm install
cp .env.example .env
npm start
```
